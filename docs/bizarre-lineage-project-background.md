# Bizarre Lineage 项目背景文档

## 1. 项目概述

本项目旨在构建一个围绕 **Roblox 游戏《Bizarre Lineage》** 的 SEO
内容站点，主要提供：

-   游戏攻略（Guides）
-   数据库（Stands / Items / Builds）
-   Tier List
-   Codes（兑换码）
-   地图与 NPC 信息
-   游戏更新资讯

目标是通过 **搜索引擎 SEO 获取 Roblox 玩家流量**，建立一个类似游戏 Wiki
的内容站。

目标站点类型： Game Wiki + Guide Hub + Database

目标用户： - Roblox 玩家 - Bizarre Lineage 新手玩家 -
查询游戏攻略的玩家 - 查询 Stands / Items / Builds 的玩家

# 3. 游戏玩法概览

Bizarre Lineage 是一个带有 Roguelike 元素的 Roblox RPG 游戏。

核心玩法包括：

### Stand 系统

玩家可以获得不同的 Stand（替身能力）。

常见 Stand 示例： - Star Platinum - The World - Whitesnake - King
Crimson - Made in Heaven

### 升级与成长

玩家通过击败 NPC、Boss 或完成任务获得经验。

成长内容包括： - 等级 - 技能 - Build

### Item 系统

游戏内存在大量道具，例如：

-   Arrow
-   Requiem Arrow
-   Rokakaka
-   Stone Mask
-   Stand Disk

这些物品通常用于： - 获得能力 - 重置能力 - 进阶能力

### Build 系统

玩家可以组合不同：

-   Stand
-   Fighting Style
-   属性

形成不同的战斗 Build，例如：

-   PvP Build
-   Farming Build
-   Tank Build

### 地图探索

游戏地图中存在：

-   NPC
-   Boss
-   Item Spawn 点

因此玩家经常搜索：

NPC locations\
Boss locations\
Item spawn locations
