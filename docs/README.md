### 当前已确认决策

- 站点定位为 `Game Wiki + Guide Hub + Database`，但 MVP 只做少数关键页面。
- 优先服务英文搜索流量，不恢复登录、支付、AI、管理后台等已禁用模块。
- 首批页面目标不是“覆盖全”，而是验证：
  - 用户是否会点击
  - 页面是否能进入索引
  - 内容是否能获得长尾流量
  - 页面模板是否值得复制扩张
